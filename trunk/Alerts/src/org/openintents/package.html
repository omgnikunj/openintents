Provides classes, interfaces, Intents, and ContentProviders for applications of the Android platform.

Package Specification

The content is divided into two separate entities that share the same packages:
  • OpenIntents.apk: This file is installed on the mobile phone by the end user.
  • openintents-lib.jar: This file is included by the developer in the "lib" directory of an Android application.
