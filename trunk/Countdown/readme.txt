The OpenIntents Countdown lets you manage several countdowns.
A notification (optionally with sound and vibration) pops up when the
countdown finished.

To obtain the current release, visit
  http://www.openintents.org


---------------------------------------------------------
release: 1.0.1
date: 2008-11-11

- Include Start button on list of countdowns.

---------------------------------------------------------
release: 1.0.0
date: 2008-11-07

- First public release on Android SDK 1.0.

Features: 
- Create several countdowns.
- Each countdown can have separate title, ringtone
  and vibration.
- Notification pops up when countdown has finished.
